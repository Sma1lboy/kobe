The web dashboard's Overview grid is now keyboard-navigable: j/k (or arrow keys) walk a highlight through the cards bucket by bucket, and Enter opens the highlighted task. Unlike the rail's j/k — which switches the active task live — the Overview highlight is local, so scanning the grid never navigates away mid-triage. The highlight follows filtering (a card that's filtered out drops the highlight) and scrolls itself into view past the fold.
