**A kanban Board view in `kobe web`** — the new `/board` route lays every worktree task out as cards on status columns (`Backlog / In progress / In review / Done`, with `Error` and `Canceled` columns appearing only when occupied). Cards carry the live engine-activity dot, PR chip, uncommitted ±counts, branch and last-update time, and clicking a card jumps straight into that task's workspace. Reachable from the top bar's board icon or the command palette's `Open board`.
