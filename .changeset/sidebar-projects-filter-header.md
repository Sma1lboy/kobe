The task sidebar's project filter now rides on the **PROJECTS** section header instead of a separate line above it. The active filter (`all` or a project name) and its matching task count sit inline on the same row — `PROJECTS  all ──── 2 tasks` — and the whole header stays clickable to cycle the filter, saving a row of vertical space in the rail.
