The web dashboard can notify you when a task needs attention: opt in from Settings → Notifications and you get a desktop notification the moment a task's engine transitions into "needs input" or "errored" while the tab is in the background — click it to jump straight to that task. Fires only on the rising edge (not every poll) and never while you're already looking at the page. Built for running many sessions at once: start them, walk away, get pinged.
