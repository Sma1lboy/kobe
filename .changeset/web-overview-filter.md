The web dashboard's Overview (mission control) gains a filter box: type to narrow the triage cards across every bucket by task title, branch, repo, path, or vendor. With many parallel sessions, find the one you mean without scanning all four columns; the header counts reflect the filtered set, and an empty result shows a "No tasks match" hint. Keyboard-first like the rail: press **/** to focus the filter and **Escape** to clear it.
